    Map의 특징

    Map은 key와 value 데이터 쌍을 갖는 자료구조 입니다. map은 공통되는 key를 가질 수 없습니다 (value는 다른 value와 겹쳐도 됨). 각각의 키는 하나의 value만 가질 수 있습니다.

    반환값 기능
    clear() void 모든 요소를 제거한다.
    containsKey(Object key) boolean 인자의 key를 갖는다면 true를 반환한다.
    containsValue(Object value) boolean 인자의 value가 하나 이상 있다면 true를 반환한다.
    entrySet() Set<Map.Entry<K,V>>
    equals(Object o) boolean o 와 비교하여 같은 map이면 true 반환한다.
    get(Object key) V 인자의 key에 해당하는 value를 반환한다.
    hashCode() int 이 map의 hashcode 를 반환한다.
    isEmpty() boolean map이 비어있는지 확인한다.
    keySet() Set<K>
